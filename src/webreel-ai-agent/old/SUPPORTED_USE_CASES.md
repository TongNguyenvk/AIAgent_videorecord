- News websites
- Blog posts
- Educational platforms

### 2. Demo Forms
- HTML forms co san trong project
- Test forms khong can backend
- UI demonstrations

### 3. E-commerce Public Pages
- Product browsing (khong can dang nhap)
- Search functionality
- Category navigation

### 4. Developer Tools
- GitHub public repos (khong can dang nhap)
- Stack Overflow search
- Package managers (npm, PyPI)

### 5. Social Media Public Content
- Public profiles
- Public posts
- Search results (khong can dang nhap)
